# Cursor Website Clone

## Introduction
This project is a clone of the Cursor website, designed to provide an intuitive interface and smooth user experience.

## Sections Implemented
1. **Home Page**: Features an overview of the services offered.
2. **About Us**: Describes the mission and vision of the Cursor.
3. **Services**: Detailed information about various services provided.
4. **Contact**: A form to reach out for more information.
5. **User Login**: Secure access for returning users.

## Design Details
- **Responsive Design**: The website is built to adapt seamlessly across devices with different screen sizes.
- **Color Scheme**: A modern color palette is used to enhance usability and aesthetics.
- **Typography**: Clear and legible fonts have been chosen to improve readability.

## Project Structure
```
├── index.html            # Home page
├── about.html            # About Us page
├── services.html         # Services page
├── contact.html          # Contact page
├── login.html            # User Login page

│   └── styles.css        # Main styles
├── js                    # Folder for JavaScript files
│   └── scripts.js        # Main scripts
└── images                # Folder for images
    ├── logo.png         # Logo image
    └── banner.jpg       # Banner image
```  

## Conclusion
This repository hosts a project that aims to replicate the Cursor website and offers necessary functionalities to demonstrate web development skills.
